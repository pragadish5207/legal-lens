# Legal-Lens Pro ⚖️

**[Live Preview](https://legal-lens-dun.vercel.app)** | An AI-powered legal companion designed to make contract analysis and legal aid highly accessible across India.

## 🚀 Overview

Legal-Lens Pro utilizes advanced AI to break down complex legal jargon into understandable insights. Whether you need a quick health check on a contract or require access to regional legal authorities, this tool bridges the gap between complex law and the everyday user.

## ✨ Key Features

* **Gamified Risk-O-Meter:** Instantly analyze contracts for potential risks with an easy-to-understand visual health check.
* **Massive Multilingual Support:** Breaking language barriers with translation capabilities for over 50+ regional Indian languages.
* **Pan-India Legal Directory:** Fully integrated with a directory of 750+ District Legal Services Authorities (DLSAs) to connect users with real legal aid.

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **AI Engine:** Gemini 1.5 Flash API
* **Hosting/Deployment:** Vercel

## 💻 Local Setup

To run this project locally on your machine:

1. Clone the repository:
   `git clone https://github.com/pragadish5207/legal-lens.git`
2. Navigate to the project directory:
   `cd legal-lens`
3. Install dependencies:
   `npm install`
4. Create a `.env` file in the root directory and add your Gemini API key:
   `REACT_APP_GEMINI_API_KEY=your_api_key_here`
5. Start the development server:
   `npm start`
